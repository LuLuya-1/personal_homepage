# Academic Personal Homepage

This project is a simple static personal homepage built with HTML and CSS.

## GitHub Pages deployment

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to Settings → Pages.
4. Under "Build and deployment", choose "GitHub Actions".
5. The workflow in [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml) will publish the site automatically.

The site will be available at:

https://<your-username>.github.io/<repository-name>/
